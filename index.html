001 — How it works

Three signals.
One quiet guard.

LoopShield runs locally in your browser. It reads your prompt input the same way you read it. No API keys. No data sent anywhere. No account required for the free version.

01

Detects similar prompts

If you re-send the same fix three times in 15 minutes, LoopShield flags it. The signal is local: word overlap, intent keywords, time window.

jaccard ≥ 0.55 + retry intent
02

Warns before you double-down

A small toast slides in. Suggested moves: stop, read the last AI reply, check the console, narrow the scope of the next prompt, or roll back instead of patching.

on-screen, dismissable
03

Snapshots before each gamble

One click saves the visible code to local storage with a timestamp. Restore by copy-paste. No GitHub setup needed. No login. Yours.

chrome.storage.local · up to 20 snaps

Questions worth asking.

Will Lovable, Bolt or v0 ban me for using this?

No. LoopShield never automates anything on your behalf. It reads your own input field the same way the page does. No requests are made to the platform's servers. It's the digital equivalent of you having a sticky note next to the screen.

Is my code or my prompts sent anywhere?

No. Everything runs in your browser. Snapshots and prompt history live in chrome.storage.local. The free version has no account, no telemetry. Pro uses ExtensionPay for licensing, which only checks a license key.

Why one-time payment instead of a subscription?

Subscriptions are how SaaS bleeds you slowly. LoopShield saves you money, it shouldn't cost you more every month. Pay once, own it.

What if my platform isn't supported?

Email hello@loopshield.app and we'll add it. Replit, Cursor web, Figma Make and Base44 are on the shortlist for v1.1.
